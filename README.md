# 🌌 SingulAI — Landing Page Oficial

Landing page moderna e responsiva para o projeto **SingulAI**, desenvolvido com **HTML, TailwindCSS, JS Vanilla** e integração nativa com **Netlify Forms**.

---

## 🚀 Visite o site

🔗 [https://singulai.site](https://singulai.site)

---

## 🧩 Tecnologias utilizadas

- ✅ HTML5 sem dependência de frameworks
- ✅ TailwindCSS 2.x via CDN
- ✅ JavaScript Vanilla para funcionalidades
- ✅ Responsividade completa (mobile-first)
- ✅ Dark Mode e Light Mode (com persistência)
- ✅ Tradução PT-BR / EN com `localStorage`
- ✅ Formulário de inscrição com Netlify Forms
